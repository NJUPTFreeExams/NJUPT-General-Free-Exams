---
title: 问题与建议提交指南
---

您可以通过 GitHub 议题（Issue）向我们提出问题或给出建议。

按照下面的步骤，以提交一个问题，指出问题或提出更改意见：

1. 前往您想要提交问题的仓库首页，点击上方的 `Issues` 按钮。
2. 在下个页面，点击右侧绿色的 `New issue` 按钮。
3. 在下个页面，填写标题和描述。

标题应该简洁清晰地概括问题，描述则应该细致地描述您的问题，可以参考 [`FIX: 文件 / 计算机通信与网络 / 2017-2018-2.pdf 中可能存在内容错误`](https://github.com/NJUPTFreeExams/NJUPT-CS-Free-Exams/issues/20)。

也可以参考 GitHub 的官方文档：

- [创建议题 - GitHub 文档](https://docs.github.com/zh/issues/tracking-your-work-with-issues/using-issues/creating-an-issue)
